yafa，转写自雅法（Jaffa）。最早的印象里，是印刷物中介绍了一种橙子。在[《纸的文化史》](https://scillidan.github.io/BYYA-site/docs/lyra-b/%E7%BA%B8%E7%9A%84%E6%96%87%E5%8C%96%E5%8F%B2/)中，也有一段描述。

未成文的文本通常在[cheat](https://github.com/scillidan/cheat)文件夹里，而`cheat`的一大部分也是[dotfile](https://github.com/scillidan/dotfile)、[Shell](https://github.com/scillidan/Shell)等文件夹的“稍后读、待办、笔记、预览”等内容。  
目前为止，写文的附带目的包括了：初次尝试、投稿、边学边写、回收、存档等等。而且经叙述时，它们总是线性的。这样，就避免文档式文本文件之间的结构、顺序、交叉引用等琐事。所以，我打算在特别清闲时将`cheat`从Web还原到文件——符合我的需求，我本来也使用fzf、ripgrep等软件来检索文件或内容——同时进行清理。那时，再考虑挖新坑。对我来说，只是从文档结构转换到了线性叙述，课题也大抵是我有使用的基础工具，如Windows平台的`cmder`、`keyprinha`等等。
