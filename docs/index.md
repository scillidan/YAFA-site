yafa，转写自雅法（Jaffa）。最早的印象里，是印刷物中介绍了一种橙子。在[《纸的文化史》](https://scillidan.github.io/BYYA-site/docs/lyra-b/%E7%BA%B8%E7%9A%84%E6%96%87%E5%8C%96%E5%8F%B2/)中，也有一段描述。

PS: 未成文的文本通常在[cheat](https://github.com/scillidan/cheat)（笔记、小抄）文件夹里，而`cheat`的一大部分也是[dotfile](https://github.com/scillidan/dotfile)、[dotfile.win](https://github.com/scillidan/dotfile.win)、[Shell](https://github.com/scillidan/scripts)等文件夹的“稍后读、待办、笔记、预览”等内容。  
目前为止，写文的目的附带了“尝试、投稿、边学边写、回收、存档”等等。而且它们在经叙述时，总归是线性的——会避免文档式文本之间的“结构、顺序、交叉引用”等琐事。我有打算/正在将cheat从Web还原到文件——我本来也使用fzf、ripgrep等工具检索文件或内容——同时日常清理。
